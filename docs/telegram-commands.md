# 텔레그램 명령어 레퍼런스

## 명령어 목록

| 명령어 | 설명 | 위험도 |
|--------|------|--------|
| `/start` | 봇 연결 확인 및 도움말 | 안전 |
| `/status` | 현재 시스템 상태 조회 | 안전 |
| `/logs [N]` | 최근 N개 로그 조회 (기본 10) | 안전 |
| `/strategy [이름]` | 전략 변경 (핫 리로드) | 중간 |
| `/backtest [전략] [일수]` | 백테스트 실행 | 안전 |
| `/halt` | 매매 중단 (킬스위치 발동) | 높음 |
| `/resume` | 킬스위치 해제, 매매 재개 | 높음 |
| `/panic_sell` | 긴급 전량 시장가 매도 | 매우 높음 |

---

## 상세 설명

### `/status`
현재 시스템 상태 요약을 즉시 보고합니다.

```
📊 시스템 상태

매매 상태: 🟢 운영 중
현재 전략: momentum
총 자산: 1,234,567원
운영 모드: 모의

차단된 마켓: 없음
```

---

### `/strategy [이름]`
서버 재시작 없이 매매 전략을 변경합니다.

```
/strategy mean_reversion
→ ✅ 전략이 mean_reversion으로 변경되었습니다.
   서버 재시작 없이 즉시 적용됩니다.
```

사용 가능한 전략 목록 확인:
```
/strategy
→ 사용 가능한 전략: momentum, mean_reversion
```

---

### `/halt`
모든 신규 주문을 즉시 차단합니다. 이미 진입된 포지션은 유지됩니다.
텔레그램에서 확인 버튼을 눌러야 실제 발동됩니다.

```
⚠️ 매매를 중단하시겠습니까?
[✅ 확인 - 매매 중단] [❌ 취소]
```

---

### `/resume`
킬스위치를 해제하고 매매를 재개합니다.

```
/resume
→ ✅ 킬 스위치 해제 완료. 매매를 재개합니다.
```

---

### `/panic_sell`
⚠️ **가장 위험한 명령어** - 모든 보유 코인을 즉시 시장가로 매도합니다.

시장의 비정상적 폭락이 예상될 때만 사용하세요.
확인 버튼을 눌러야 실제 실행됩니다.

```
🚨 긴급 전량 매도
⚠️ 모든 보유 코인을 즉시 시장가로 매도합니다.
이 작업은 되돌릴 수 없습니다!

[🚨 전량 매도 실행] [❌ 취소]
```

---

## 자동 알림

다음 이벤트에서 자동으로 알림이 전송됩니다:

- **킬스위치 발동**: 포트폴리오 낙폭이 임계치 초과 시
- **손절 발동**: 개별 코인이 손절 기준선 하락 시
- **수동 halt 발동**: 텔레그램 명령으로 중단 시
- **에러 발생**: 시스템 치명적 오류 시 (1시간 내 중복 억제)

## 보안

- 봇은 `.env`의 `TELEGRAM_CHAT_ID`에 등록된 채팅 ID만 응답합니다
- 다른 사용자가 명령을 보내면 무시되고 보안 로그에 기록됩니다
- 봇 토큰을 절대 공개 저장소에 올리지 마세요
